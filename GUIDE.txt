Тут расписано где и что менять в декомпилированных исходниках.

Строки зашиты в бинарник libapp.so, просто так текстовик не поменяешь.
Чтобы реально изменить поведение, нужно патчить бинарь через hex-редактор
или пересобирать APK из восстановленного Dart-проекта.

Ниже описано ГДЕ в декомпиле находится каждая строка, чтобы понимать
что за что отвечает и что именно патчить.

2. ССЫЛКА КНОПОК "TELEGRAM" И "ПОПОЛНИТЬ СЧЁТ"
-----------------------------------------------------------

//Автор - @ted_resolution

Обе кнопки ведут на один и тот же URL. При нажатии вызывается
функция launch() из пакета url_launcher, которая открывает ссылку:

  https://t.me/crmp_rage

Где она прописана:

Файл: dart_decompiled/asm/url_launcher/src/legacy_api.dart
Строка 22-23:  r1 = "https://t.me/crmp_rage"
  >> основной вызов, отсюда url_launcher берет ссылку

Файл: dart_decompiled/pp.txt (Object Pool - таблица всех строк)
Строка 12166: [pp+0xd950] String: "https://t.me/crmp_rage"
  >> смещение в object pool: 0xd950

Файл: dart_decompiled/asm/ragerussia/features/app/presentation/widgets/start_page.dart
Строки 120-121:   r0 = "TELEGRAM"
  >> текст первой кнопки
Строки 237-239:   r0 = "ПОПОЛНИТЬ СЧЁТ"
  >> текст второй кнопки
Строка 1229-1230: r0 = launch()
  >> вызов открытия ссылки при нажатии (обе кнопки юзают одну closure
     которая вызывает launch с захардкоженным URL)

Чтобы изменить ссылку, надо патчить строку "https://t.me/crmp_rage"
в бинарнике libapp.so по смещению из object pool (pp+0xd950).

3. ССЫЛКА СКАЧИВАНИЯ АПК (кнопка "ЗАГРУЗИТЬ" / "УСТАНОВИТЬ")
-----------------------------------------------------------

//Автор - @ted_resolution

При нажатии кнопки "ЗАГРУЗИТЬ" лаунчер качает два файла:
  - сам лаунчер (обновление)
  - игровой клиент

URLs зашиты в bloc.dart, вот точные места:

Файл: dart_decompiled/asm/ragerussia/features/app/presentation/bloc/bloc.dart

  Строка 1402-1403:  "https://fastdl.ragerussia.online/version.json"
    >> проверка версии (json с актуальной версией)

  Строка 1406-1407:  "https://fastdl.ragerussia.online/launcher.apk"
    >> скачивание APK лаунчера (game launcher)

  Строка 1410-1411:  "https://fastdl.ragerussia.online/launcherMain.apk"
    >> скачивание главного APK (сам лаунчер для обновления)

  Строка 1999-2000:  "https://fastdl.ragerussia.online/launcherMain.apk"
    >> повторное использование - скачивание + открытие через OpenFile

  Строка 2209-2210:  "https://fastdl.ragerussia.online/launcher.apk"
    >> повторное использование - скачивание + открытие через OpenFile

Локальные имена файлов после скачки:

  Строка 1394-1395:  "game.apk"       — имя скачанного игрового клиента
  Строка 1398-1399:  "launcher.apk"   — имя скачанного лаунчера

Смещения в Object Pool (для патча бинаря):

Файл: dart_decompiled/pp.txt
  Строка 8170:  [pp+0x7a30] "https://fastdl.ragerussia.online/version.json"
