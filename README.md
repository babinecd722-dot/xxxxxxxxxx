# Привет

Этот репозиторий содержит скрипты и утилиты для SA-MP (San Andreas Multiplayer).

## Содержимое

- **aim_fix_updated.lua** — скрипт для фикса синхронизации прицела (AimSync FIX, автор: Ulong)
- **send_ping_fix.lua** — скрипт для фикса отправки пинга (автор: Ulong)
- **RakNet / RakSAMP** — библиотеки для работы с сетевым протоколом RakNet
- **BpcEditor** — редактор BPC-файлов
- **GUIDE.txt** — краткий гайд по кастомизации лаунчера
